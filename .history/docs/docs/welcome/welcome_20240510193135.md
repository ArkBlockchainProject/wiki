# 生为民不谋利
伯纳斯先生最先提出了万维网的概念，经过近半个世纪的发展，逐渐成熟，以我们无法想象的速度快速迭代，直到我们今天所看到的先阶段的最终形态Web 3。Web 3与Web 2一样，**它们有一个共同的特征就是为所有人开放**,并且您可以像从初代互联网迁移到Web 2一样方便。每一代的更新都带来了众多与众不同的新功能，例如用富文本替换了超链接，而我们眼前的Web 3，带来了一个至关重要的内容：分布式账本与金融。它与传统的中心化金融有许多相似的地方，但是它的出现会大大改变我们的生活。

这是Firefox的一句话，我们认为放在这里很合适。这份知识库是为所有人打造的，**我们不会植入广告或者推出付费申请，同时我们保护您的隐私，除了必要的分析工具外，并不会获取更多信息。**
