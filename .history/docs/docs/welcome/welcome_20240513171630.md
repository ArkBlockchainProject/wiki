# 简介
这是一份有关 Web 3 一切

关于 Web 3 的定义，我们无从得知。但是我们可以给出一个笼统的观点：**那就是去中心化、透明与通证时代。**。
- 去中心化意味着那些不可信的机构将不复存在，并且任何内容都没有一个掌管大全的中心节点。
- 透明意味着隐私的透明。我们坚信隐私与透明是一切的根基。
- 通证时代可以简要地理解为一种表现形式，通证的出现意味着任何事物都可以当作通证在区块链当中实现。
