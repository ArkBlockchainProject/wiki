# 与我们一起前行
独自完成如此庞大的知识库显然是不现实的。如果您有意愿加入我们看看下面的流程。
# 要求与流程
## 如何参与？
想要参与非常简单，您只需要 Fork 本仓库，并且将代码 clone 到本地。
::: danger 请注意！
由于自动部署依赖于 Github Actions，所以您不需要修改其他地方的文件，只需要关注 docs 文件夹下的文件即可。
:::
在您修改完成后，请将代码提交到远程仓库，并提交合并请求，我们会在 24 小时内通过。
